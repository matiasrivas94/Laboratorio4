Pasos para la instalacion:
1-Clonar el poryecto a la maquina local(en donde quiera descargarlo).
	git clone + direccion del repositorio
	git clone https://github.com/matiasrivas94/Laboratorio4

2-Tener instalado mongodb, postamn y algun editor de texto(visual studio utilizado)

3-Abri las carpetas de backend y frotend por consola, introducir npm install para instalar las dependencias.

4-Ejecutar mongoDB

5-Introducir las siguientes lineas de codigo:
	A)En el backend: npm start o npm start index.js
	B)En el frontend: ng serve

6-Listo para su uso!
